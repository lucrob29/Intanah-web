# Intanah Website

A modern, responsive website for Intanah, showcasing wedding and event venues in Bali.

## Features

- Responsive design
- Video backgrounds
- Interactive elements
- Business model showcase
- Project portfolio
- Contact information

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start editing!

## Project Structure

```
intanah-website/
├── index.html
├── styles.css
├── script.js
├── images/
├── videos/
└── README.md
```
